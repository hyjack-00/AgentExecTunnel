# Evaluation v0.1.1

## Changes

- `.gitmodules` removed; `agent_forward/` and `agent_backward/` added to `.gitignore`
- new module `agent_exec_tunnel/remotes.py` centralizes data-repo remote resolution
- `tools/bootstrap_repos.py` rewritten to clone-or-sync the two data repos directly
- `tools/run_burst_local_relay.py` and `tools/run_burst_live.py` now resolve remotes via `agent_exec_tunnel.remotes.load_remotes`
- `tests/test_fresh_clone.py` drops submodule plumbing and injects remotes via env vars
- `tests/test_cli_entrypoints.py` replaces the `ensure_repo_local_origin` test with clone-and-rewire coverage for `ensure_repo`
- `README.md` Startup section rewritten; submodule references removed
- `PROGRESS.md` updated with new TODO entries and notes about the remote resolution chain

## Coverage

- unit: `tests/test_cli_entrypoints.py::CliEntrypointTests::test_bootstrap_ensure_repo_clones_from_remote_when_missing`

- integration: `tests/test_fresh_clone.py::FreshCloneTests::test_bootstrap_and_executor_once_from_fresh_clone_layout`
